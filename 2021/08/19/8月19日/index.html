git：码云，github，gitlab，coding+腾讯云

怎么使⽤git

git clone 地址 : 克隆⼀个仓库地址

git init : git 初始化 ⽣成 .git⽂件

git add ⽂件名字 : git add . : 将所有⽂件提交到缓存 没有提交记录

git commit -m ‘描述信息’ : 提交到本地

git log : 提交⽇志

git status : 查看git状态

git remote add origin 地址 : 关联远程地址

git remote -v : 查看有没有关联远程地址

git push origin 分⽀ : 推送到远程

git pull : 从远程拉取

git pull origin 分⽀名字

# 团队协作 : 分⽀

默认分⽀(主分⽀) : master

合并 : dev分⽀

git branch 王佳鑫

git checkout 董贞

git merge 分⽀名字 : 合并分⽀

\2. 其中⼀个⼈都克隆 写点代码 提交上去 master

\4. A 写代码

\10. 提交到远程

\11. A 拉取
